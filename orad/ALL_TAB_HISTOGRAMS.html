ALL_TAB_HISTOGRAMS

Histograms on columns of all tables visible to user 

Columns
   ___________________________
 
   OWNER
      Owner of table
   TABLE_NAME
      Table name
   COLUMN_NAME
      Column name or attribute of object column
   ENDPOINT_NUMBER
      Endpoint number
   ENDPOINT_VALUE
      Normalized endpoint value
   ENDPOINT_ACTUAL_VALUE
      Actual endpoint value

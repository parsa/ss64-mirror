Year

Return a four-digit year (a number from 1900 to 9999) for a given date value.

Syntax
      Year(date_value)

Key
   date_value  A valid date.

Examples

Dim intDemo
intDemo = Year(dtmShiftEnd)
intDemo = Year(#31/12/2012#)

Select * from orders where Year(Order_date) = 2012;
